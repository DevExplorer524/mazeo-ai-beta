# Mazeo AI v2.0 - Deep Learning Edition

Mazeo is a state-of-the-art AI assistant created by **Mazen Saber**. 

## New Features in v2.0:
- **Local Deep Learning**: Powered by a custom NumPy-based neural network for sentiment and intent analysis.
- **Massive Knowledge Base**: Integrated system data covering physics, math, and technology.
- **Real-time Awareness**: Aware of the current date, time, and world context.
- **Hybrid Intelligence**: Combines local processing with advanced cloud LLM capabilities.
- **Fluent Arabic Support**: Enhanced natural language processing for Arabic users.

## How to Run:
1. Run `run_mazeo.bat` to install dependencies and start the server.
2. Open your browser to `http://127.0.0.1:5000`.
